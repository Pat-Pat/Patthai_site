Opensource : Tracky Colour

Tracky is a software that connect human gesture to computer system by tracking marker color. This software was developed in 24 hr at non-stop coding event "Codeday" at portland 2014. Tyler, Adam, and Hamza for you awesome codeday, and finally I would like to thank my friends in tech team, my PSU.WIT Teacher, and my family who gave me an opportunity to pursue coding :)

Download

There are several options for download. The source code is Java based and need to be run on Processing. The UI was developed in Swishmax.
